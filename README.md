# react-native-hooks-e-context-api-cod3r
Aplicativo em React Native desenvolvido em um dos módulos do curso da plataforma COD3R(https://www.cod3r.com.br/courses/react-native-crud), tendo foco no desenvolvimento de um CRUD de usuários utilizando Hooks e Context API

Para compilar e visualizar o aplicativo os seguintes passos devem ser seguidos:
'npm install' ou 'yarn install'
'npx react-native start'

Após os comandos acima, deve-se abrir o emulador ou conectar o smartphone via aplicativo Expo ou via cabo USB diretamente no computador e utilizar o comando abaixo:
'npx react-native run-android' ou 'npx react-native run-ios'

Tutorial mais explicativo e detalhado: https://reactnative.dev/docs/running-on-device
